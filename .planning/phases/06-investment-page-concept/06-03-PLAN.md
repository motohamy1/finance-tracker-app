**Component interface:**
```typescript
interface BottomSheetProps {

  onClose: () => void;

```

**Implementation details:**

**Animation:** Use react-native-reanimated. When `visible` becomes true:
- Slide the sheet from below the screen (translateY 300 → 0) using `withTiming` with duration 250ms and `withSpring` on close.
- Fade in a semi-transparent backdrop (opacity 0 → 1).
- When `visible` becomes false: reverse the animations.

**Structure:**
```jsx
<Modal visible={visible} transparent animationType="none" onRequestClose={onClose}>
  <Animated.View style={[styles.backdrop, backdropStyle]}>
    <TouchableWithoutFeedback onPress={onClose}>
      <View style={{ flex: 1 }} />
    </TouchableWithoutFeedback>
  </Animated.View>
  <Animated.View style={[styles.sheet, sheetStyle]}>
    {/* Drag handle — a small gray pill at top */}
    <View style={styles.handleContainer}>
      <View style={styles.handle} />
    </View>
    {/* Optional title */}
    {title && <Text style={styles.title}>{title}</Text>}
    {/* Content */}
    {children}
  </Animated.View>
</Modal>
```

**Styling:**
- Backdrop: backgroundColor 'rgba(0,0,0,0.4)', absolute fill
- Sheet: backgroundColor '#FFFFFF', borderTopLeftRadius 20, borderTopRightRadius 20, paddingBottom 34 (safe area), paddingHorizontal 16, paddingTop 8
- Handle: width 40, height 4, borderRadius 2, backgroundColor '#E2E8F0', alignSelf 'center', marginBottom 16
- Title: fontSize 18, fontWeight '700', color '#0F172A', textAlign 'center', marginBottom 16

**Animation implementation:**
Use two shared values: `backdropOpacity = useSharedValue(0)` and `sheetTranslateY = useSharedValue(300)`.

Use `useEffect` to trigger animations when `visible` changes:
```typescript
useEffect(() => {
  if (visible) {
    backdropOpacity.value = withTiming(1, { duration: 200 });
    sheetTranslateY.value = withTiming(0, { duration: 250 });
  } else {
    backdropOpacity.value = withTiming(0, { duration: 200 });
    sheetTranslateY.value = withTiming(300, { duration: 200 });
  }
}, [visible]);
```

```typescript

}));
```

    activeOpacity={0.7}
  >
    <Text style={[styles.chipText, !selected && styles.chipTextActive]}>All</Text>
