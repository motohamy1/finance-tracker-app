    IMPLEMENTATION DETAILS:

    **exportTableToJSON(table):**
    - Switch on table name to call the correct getAll function:
      - 'categories' → `getAllCategories()` → map to camelCase objects
      - 'expenses' → fetch all expenses: first get all categories, then `getExpensesByCategory(id)` for each, flatten results
      - 'trades' → `getAllTrades()`
      - 'current_prices' → `getAllCurrentPrices()` → convert Record to array of `{ ticker, priceCents, updatedAt }`
    - Return the array directly (caller handles serialization)

    **syncAll():**
    1. Check authentication: `const authed = await isAuthenticated(); if (!authed) return { success: false, error: 'Not authenticated' }`
    2. Download remote files:
       - Fetch manifest via `downloadManifest()`
       - List all files via `listSyncFiles()`
       - For each table file found, download via `downloadJSON<T>(fileId)`
       - If no remote files exist, skip merge, just upload local data
    3. For each SYNC_TABLE:
       a. Get local data via `exportTableToJSON(table)`
       b. Get remote data (from step 2 download, empty array if not found)
       c. Perform LWW merge (D-05):
          - Build Map<id, record> from local array
          - For each remote record:
            - If id exists in local AND remote.updatedAt > local.updatedAt → replace local with remote
            - If id exists in local AND local.updatedAt >= remote.updatedAt → keep local
            - If id does NOT exist in local → add remote (it was created on another device)
          - For local records not in remote → keep (they're new local records)
          - The merged result becomes the new truth for both local DB and Drive
       d. Save merged data to local DB:
          - For each table, DELETE ALL existing rows first, then INSERT the merged array
          - Use transactions: `BEGIN TRANSACTION; DELETE FROM {table}; INSERT ...; COMMIT;`
          - For categories: `createCategory(id, name, colorHex, sortOrder)` — but we need to preserve created_at/updated_at from original. Create a raw insert function or use `INSERT OR REPLACE`
          - SIMPLER APPROACH: Use `INSERT OR REPLACE INTO {table} (col1, col2, ...) VALUES (?, ?, ...)` for each record. This preserves all columns including created_at/updated_at.
          - Use `db.runSync()` in a transaction for atomicity
       e. Record mapping for Drive file IDs (from step 2)
    4. Upload merged data to Drive:
       - For each table, `uploadJSON(table, mergedArray, existingFileId)`
    5. Create/update manifest:
       - `uploadManifest({ version: 1, tables: {...}, deviceId: generateUUID(), updatedAt: new Date().toISOString() }, existingManifestId)`
    6. Update sync log in AsyncStorage:
       - For each table, set `SyncLog { table, lastSyncAt: now, lastSyncRecordCount: mergedArray.length }`
       - Save to AsyncStorage under SYNC_LOG_KEY as JSON
    7. Return `{ success: true }`

    IMPORTANT: The LWW merge MUST use the `updated_at` / `updatedAt` field. All 4 tables have this column. The comparison is string-based ISO 8601 timestamp comparison (lexicographic = chronological for ISO format).

    IMPORTANT: For `expenses`, we need a `getAllExpenses()` function. Since the existing codebase doesn't have one, add it to the database service inline (export it from sync.ts's implementation or add a helper):
    ```typescript
    // Helper to get all expenses across all categories
    function getAllExpenses(): Expense[] {
      const categories = getAllCategories();
      return categories.flatMap(c => getExpensesByCategory(c.id));
    }
    ```
    This avoids modifying database.ts unnecessarily.

    **restoreAll():**
    1. Check authentication
    2. Call listSyncFiles() to find available files
    3. For each SYNC_TABLE file found, download via downloadJSON()
    4. Import each table's data using the same INSERT OR REPLACE transaction pattern as syncAll step 3d
    5. Count total records imported across all tables
    6. Set restorePromptDismissed in AsyncStorage
    7. Return `{ success: true, recordCount: N }`

    **checkRestoreAvailable():**
    1. Check AsyncStorage for RESTORE_DISMISSED_KEY — if already dismissed, return false
    2. Check authentication
    3. Call listSyncFiles()
    4. If any sync files exist (check for at least 'sync/manifest.json' or any 'sync/*.json'), set RESTORE_FLAG_KEY = 'true' in AsyncStorage, return true
    5. Otherwise return false

    **getLastSyncTimes():**
    - Read SYNC_LOG_KEY from AsyncStorage
    - Parse JSON → SyncLog[]
    - Return array (empty if no data)

    **Direct database import (INSERT OR REPLACE) approach:**
    For the LWW merge save-back, use the `getDatabase()` accessor from `@/db/schema`:
    ```typescript
    import { getDatabase } from '@/db/schema';
    
    function importTable(table: SyncTable, records: Record<string, unknown>[]): void {
      const db = getDatabase();
      db.execSync('BEGIN TRANSACTION;');
      try {
        db.runSync(`DELETE FROM ${table};`);
        for (const record of records) {
          // Map camelCase TS keys to snake_case DB columns
          const columns = Object.keys(record).map(keyToSnakeCase).join(', ');
          const placeholders = Object.keys(record).map(() => '?').join(', ');
          const values = Object.values(record);
          db.runSync(`INSERT OR REPLACE INTO ${table} (${columns}) VALUES (${placeholders});`, values);
        }
        db.execSync('COMMIT;');
      } catch (e) {
        db.execSync('ROLLBACK;');
        throw e;
      }
    }
    ```

    NOTE: The `keyToSnakeCase` helper must convert camelCase TS keys (e.g., `categoryId`, `amountCents`, `pricePerShareCents`) to snake_case DB columns (e.g., `category_id`, `amount_cents`, `price_per_share_cents`). Create this as a private helper inside sync.ts:
    ```typescript
    function camelToSnake(key: string): string {
      return key.replace(/[A-Z]/g, letter => `_${letter.toLowerCase()}`);
    }
    ```

    Zustand store for all cloud sync state. Follows the same pattern as existing stores (expenseStore.ts, tradeStore.ts).

    ```typescript
    import { create } from 'zustand';
    import AsyncStorage from '@react-native-async-storage/async-storage';
    import type { SyncState, SyncTable, SyncLog } from '@/types';
    import { signIn, signOut, getAccessToken, isAuthenticated, getGoogleEmail } from '@/services/auth';
    import { syncAll, restoreAll, checkRestoreAvailable, getLastSyncTimes } from '@/services/sync';

    // Initial sync logs for all 4 tables
    const INITIAL_SYNC_LOGS: Record<SyncTable, SyncLog> = {
      categories:    { table: 'categories',    lastSyncAt: '', lastSyncRecordCount: 0 },
      expenses:      { table: 'expenses',      lastSyncAt: '', lastSyncRecordCount: 0 },
      trades:        { table: 'trades',        lastSyncAt: '', lastSyncRecordCount: 0 },
      current_prices:{ table: 'current_prices',lastSyncAt: '', lastSyncRecordCount: 0 },
    };

    interface SettingsActions {
      /** Trigger Google Sign-In (D-07: opt-in — user must enable) */
      login: () => Promise<boolean>;

      /** Sign out and reset sync state */
      logout: () => Promise<void>;

      /** Toggle sync enabled/disabled (D-07) */
      setSyncEnabled: (enabled: boolean) => void;

      /** Perform a manual sync (D-09: "Sync Now" button) */
      syncNow: () => Promise<void>;

      /** Trigger restore from Drive (D-08: fresh install prompt) */
      doRestore: () => Promise<{ success: boolean; recordCount: number }>;

      /** Dismiss the restore prompt (D-08: shown only once) */
      dismissRestorePrompt: () => void;

      /** Initialize: check auth state, restore availability, load sync logs */
      initialize: () => Promise<void>;

      /** Reload sync logs from AsyncStorage */
      refreshSyncLogs: () => Promise<void>;
    }

    export const useSettingsStore = create<SyncState & SettingsActions>((set, get) => ({
      // ─── State (initial values — sync disabled by default per D-07) ───
      isAuthenticated: false,
      isSyncEnabled: false,
      isSyncing: false,
      lastSyncAt: null,
      lastSyncError: null,
      googleEmail: null,
      restoreAvailable: false,
      restorePromptDismissed: false,
      syncLogs: { ...INITIAL_SYNC_LOGS },

      // ─── Actions ───
      
      initialize: async () => {
        // 1. Check if already authenticated (token in SecureStore)
        const authed = await isAuthenticated();
        // 2. Get email if authenticated
        const email = authed ? await getGoogleEmail() : null;
        // 3. Check if restore available (fresh install detection)
        const restoreAvail = authed ? await checkRestoreAvailable() : false;
        // 4. Load sync logs
        const logs = await getLastSyncTimes();
        const syncLogs = { ...INITIAL_SYNC_LOGS };
        for (const log of logs) {
          syncLogs[log.table] = log;
        }
        // 5. Determine lastSyncAt from logs (most recent across all tables)
        const lastSyncAt = logs.length > 0
          ? logs.reduce((a, b) => a.lastSyncAt > b.lastSyncAt ? a : b).lastSyncAt
          : null;

        set({
          isAuthenticated: authed,
          googleEmail: email,
          restoreAvailable: restoreAvail,
          syncLogs,
          lastSyncAt,
        });
      },

      login: async () => {
        const result = await signIn();
        if (result) {
          const email = await getGoogleEmail();
          set({ isAuthenticated: true, googleEmail: email, lastSyncError: null });
          // Check for restore after login
          const restoreAvail = await checkRestoreAvailable();
          set({ restoreAvailable: restoreAvail });
        }
        return result;
      },

      logout: async () => {
        await signOut();
        set({
          isAuthenticated: false,
          isSyncEnabled: false,
          googleEmail: null,
          lastSyncAt: null,
          restoreAvailable: false,
          restorePromptDismissed: false,
          syncLogs: { ...INITIAL_SYNC_LOGS },
          lastSyncError: null,
        });
      },

      setSyncEnabled: (enabled: boolean) => {
        set({ isSyncEnabled: enabled });
      },

      syncNow: async () => {
        const { isSyncing, isAuthenticated } = get();
        if (isSyncing || !isAuthenticated) return;

        set({ isSyncing: true, lastSyncError: null });
        try {
          const result = await syncAll();
          if (result.success) {
            const logs = await getLastSyncTimes();
            const syncLogs = { ...INITIAL_SYNC_LOGS };
            for (const log of logs) {
              syncLogs[log.table] = log;
            }
            const lastSyncAt = logs.length > 0
              ? logs.reduce((a, b) => a.lastSyncAt > b.lastSyncAt ? a : b).lastSyncAt
              : new Date().toISOString();
            set({ lastSyncAt, syncLogs, isSyncing: false });
          } else {
            set({ lastSyncError: result.error || 'Sync failed', isSyncing: false });
          }
        } catch (e: unknown) {
          set({ lastSyncError: e instanceof Error ? e.message : 'Unknown sync error', isSyncing: false });
        }
      },

      doRestore: async () => {
        set({ isSyncing: true, lastSyncError: null });
        try {
          const result = await restoreAll();
          if (result.success) {
            set({
              isSyncing: false,
              restoreAvailable: false,
              restorePromptDismissed: true,
              lastSyncAt: new Date().toISOString(),
            });
          } else {
            set({ lastSyncError: result.error || 'Restore failed', isSyncing: false });
          }
          return { success: result.success, recordCount: result.recordCount };
        } catch (e: unknown) {
          const error = e instanceof Error ? e.message : 'Unknown restore error';
          set({ lastSyncError: error, isSyncing: false });
          return { success: false, recordCount: 0 };
        }
      },

      dismissRestorePrompt: () => {
        set({ restorePromptDismissed: true, restoreAvailable: false });
      },

      refreshSyncLogs: async () => {
        const logs = await getLastSyncTimes();
        const syncLogs = { ...INITIAL_SYNC_LOGS };
        for (const log of logs) {
          syncLogs[log.table] = log;
        }
        set({ syncLogs });
      },
    }));
    ```

    - Use `get()` to access current state inside actions (not closures over mutable values)
    - Async actions use try/catch with error → `lastSyncError` state
    - The `initialize()` pattern matches expenseStore's approach
